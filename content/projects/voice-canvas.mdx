## Overview

Speak into your microphone and watch your voice transform into a 3D landscape of light, color, and motion. Voice Canvas analyzes your speech in real-time — frequency, amplitude, timbre — and maps those properties to visual parameters.

Whisper and the landscape gently ripples. Shout and it erupts into mountains. Sing and it blooms with color.

## Technical Architecture

The audio pipeline uses the Web Audio API's `AnalyserNode` to extract frequency data at high resolution:

```typescript
const audioContext = new AudioContext();
const analyser = audioContext.createAnalyser();
analyser.fftSize = 2048;

const source = audioContext.createMediaStreamSource(microphoneStream);
source.connect(analyser);

// Extract meaningful features from raw frequency data
function extractFeatures(dataArray: Float32Array): VoiceFeatures {
  const fundamental = findFundamentalFrequency(dataArray);
  const amplitude = dataArray.reduce((sum, v) => sum + Math.abs(v), 0) / dataArray.length;
  const spectralCentroid = calculateSpectralCentroid(dataArray);
  const harmonics = extractHarmonics(dataArray, fundamental);

  return { fundamental, amplitude, spectralCentroid, harmonics };
}
```

These features drive the 3D visualization built with Three.js and custom GLSL shaders:

- **Fundamental frequency** → terrain height
- **Amplitude** → camera distance and bloom intensity
- **Spectral centroid** → color temperature
- **Harmonics** → particle density and behavior

## Visual Modes

1. **Landscape** — Your voice becomes terrain. Conversation creates rolling hills; singing creates dramatic peaks.
2. **Nebula** — Particles swirl and cluster based on vocal characteristics. Multiple speakers create interacting galaxy systems.
3. **Abstract** — Pure color and motion, more painterly than physical.

## Challenges

- **Latency**: Getting the audio-to-visual pipeline under 16ms required careful optimization. The FFT runs on a separate thread via AudioWorklet.
- **Microphone permissions**: Users are understandably wary. The app explains clearly what's happening and never records or transmits audio.
- **Musical vs. speech input**: The visualization responds differently to singing vs. talking, which required training a simple classifier on harmonic ratios.

## What I Learned

Sound is incredibly rich data. The difference between a whisper and a shout, between singing and speaking, between one voice and two — all of these create visually distinct patterns. The human voice is an instrument for generative art.
